PROJECTS

Course Projects

Seq2Seq Models for NMT

DS 303: Introduction to Machine Learning
This project explores various sequence to sequence architectures out there for the task of neural machine transliteration. A further extension of the project includes integration of OCR techniques with the methods used to develop a pipelined method to transliterate images.

Indian Stock Market during COVID-19

 DS 203: Programming for Data Science
This project finds out the impact of COVID-19 on various sectors of the Indian Market, and proceeds to take the current scenario as an example to train various RNNs and CNNs to predict stock market outcomes in a similar event.

Arithmetic Logical Unit in VHDL

 EE 224: Digital Systems
This project implements an arithmetic logical unit (ALU) in VHDL in Quartus. The ALU performs fast addition and subtraction (16 bits) using the Kogge Stone Adder and allows XOR and NAND operations on the inputs.

Morse Code and tone

EEE 337: Microprocessors Laboratory
This project implements the Morse Code system on the PT-51 board using Assembly and Embedded-C hardware description languages. A further addition to the project was generating sounds of the correct frequencies to convert the morse text signal to audio signals / tone for a musical effect.

Autonomous Garbage Collecting Bot

ITSP, ITC, IIT Bombay
Designed an autonomous garbage collecting bot which can detect and classify three types of garbage. Modeled the bot on Blender, created the circuit on Fritzing using R-Pi4 and Arduino, and used YOLO for object detection, ROS for simulation. Declared as one of the top 3 projects out of 70+.

Non-Small Cell Lung Cancer Prediction

MeDAL Lab, IIT Bombay
Worked on detection of non-small cell lung cancer (LUAD vs LUSC) by working upon the Inception Deep Learning architecture and multi-task multi-label classification of 10 genomes causing the respective cancer type.

OBSTACLE manoeuvring BOT

Electronics and Robotics Club, IIT Bombay
Designed and built Bluetooth controlled (HC-05) robotic car with differential mechanism for turning and used IC7805 as a voltage regulator, L293D as the motor driver and H-Bridge for the robotic car.

pyfrac

 Self Project
Pyfrac is a library written (by me), completely in Python to work with rational numbers (i.e fractions). I felt that the inbuilt fractions module was insufficient since it creates immutable objects, and hence this can be considered as an extension allowing mutability, and adds many more features such as alternative constructors.

Rubik's Cube Simulator
